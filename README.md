# 🎬 FilmOrbit

FilmOrbit is a movie discovery and review website built for our **Full Stack Development Lab Project**.

The platform allows users to explore movies, read blogs, view movie galleries, and interact with different web components.

---

## 🚀 Features

- Responsive landing page
- Movie discovery section
- Blog article page
- FAQ section with toggle
- Modal popup for reviews
- Character counter in review form
- Movie poster gallery with filters
- Currency converter using API
- Dark / Light theme toggle

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- ExchangeRate API
